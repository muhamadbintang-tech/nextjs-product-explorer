# 🛒 Abinshop - Next.js Product Explorer

Aplikasi katalog produk modern yang dibangun menggunakan **Next.js 16 (App Router)**, **Tailwind CSS**, dan **TypeScript**, terintegrasi dengan **DummyJSON API**.

## 🚀 Live Demo
Website yang sudah ter-deploy dapat diakses di: **[https://nextjs-product-explorer-nu.vercel.app](https://nextjs-product-explorer-nu.vercel.app)**

## ✨ Fitur Utama & Fitur Tambahan
- 🔍 **Debounce Search**: Pencarian responsif tanpa lag.
- 📊 **Sorting**: Filter harga (Terendah / Tertinggi) & Rating.
- ❤️ **Favorites (LocalStorage)**: Simpan produk favorit secara real-time.
- 📄 **Pagination & Smooth Scroll**: Navigasi halaman dengan efek auto-scroll ke atas.
- 🔗 **URL Query Parameters**: Menyimpan pencarian & filter di URL.
- 💀 **Skeleton Loading & Error State**: Tampilan loading dan error handling yang user-friendly.
